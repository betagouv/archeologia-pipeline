================================================================================
                    TESTS MANUELS - PLUGIN ARCHEOLOG'IA PIPELINE
================================================================================

Pré-requis:
- QGIS 3.x installé
- Plugin activé dans QGIS
- PDAL, GDAL (gdalwarp, gdal_translate, gdaladdo) dans le PATH
- Fichiers de test disponibles (LAZ, MNT, RVT selon les modes)

================================================================================
1. TESTS DE CHARGEMENT DU PLUGIN
================================================================================

[ ] 1.1 Démarrage QGIS
    - Ouvrir QGIS
    - Vérifier qu'aucune erreur n'apparaît dans la console Python
    - Vérifier que le plugin apparaît dans le menu Extensions

[ ] 1.2 Ouverture du plugin
    - Cliquer sur le menu/icône du plugin
    - Vérifier que la fenêtre MainDialog s'ouvre sans erreur
    - Vérifier que tous les onglets sont visibles et accessibles

[ ] 1.3 Fermeture et réouverture
    - Fermer la fenêtre du plugin
    - Rouvrir le plugin
    - Vérifier que la configuration précédente est conservée

================================================================================
2. TESTS DE L'INTERFACE UTILISATEUR
================================================================================

[ ] 2.1 Changement de mode
    - Sélectionner chaque mode (ign_laz, local_laz, existing_mnt, existing_rvt)
    - Vérifier que les champs spécifiques au mode s'affichent correctement
    - Vérifier que le label du champ source change selon le mode

[ ] 2.2 Sélection du dossier de sortie
    - Cliquer sur le bouton de sélection du dossier de sortie
    - Sélectionner un dossier valide
    - Vérifier que le chemin s'affiche correctement

[ ] 2.3 Configuration des produits
    - Cocher/décocher chaque produit (MNT, DENSITE, M_HS, SVF, SLO, LD, VAT)
    - Vérifier que les cases à cocher fonctionnent

[ ] 2.4 Configuration des paramètres RVT
    - Modifier les paramètres de chaque onglet RVT (MDH, SVF, Slope, LDO, VAT)
    - Vérifier que les spinbox acceptent les valeurs

[ ] 2.5 Configuration Computer Vision
    - Activer/désactiver la CV
    - Sélectionner un modèle (si disponible)
    - Modifier les seuils (confidence, IOU)

================================================================================
3. TESTS DE SAUVEGARDE/CHARGEMENT CONFIG
================================================================================

[ ] 3.1 Sauvegarde manuelle
    - Modifier plusieurs paramètres
    - Cliquer sur le bouton Sauvegarder
    - Fermer et rouvrir le plugin
    - Vérifier que les paramètres sont conservés

[ ] 3.2 Reset configuration générale
    - Cliquer sur "Réinitialiser config générale"
    - Vérifier que les valeurs par défaut sont restaurées

[ ] 3.3 Reset configuration RVT
    - Cliquer sur "Réinitialiser config RVT"
    - Vérifier que les valeurs RVT par défaut sont restaurées

[ ] 3.4 Pas de sauvegarde auto sur Run
    - Modifier un paramètre
    - Cliquer sur Run (puis Cancel immédiatement)
    - Fermer et rouvrir le plugin
    - Vérifier que le paramètre modifié N'EST PAS sauvegardé

4. TESTS DU PREFLIGHT
================================================================================

[ ] 4.1 Preflight avec dépendances OK
